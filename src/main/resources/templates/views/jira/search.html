Suggerimenti:

  • Verifica che il codice del progetto sia corretto (es. "PROJECT" e non "PROJ")
  • Controlla la sintassi della query JQL
  • Assicurati di avere i permessi per accedere ai ticket

Inserisci una query JQL valida per cercare i ticket.
Esempi di Query JQL

Ecco alcuni esempi di query JQL che puoi utilizzare:

  • project = PROJ - Tutti i ticket del progetto PROJ
  • project = PROJ AND status = "In Progress" - Ticket in corso del progetto PROJ
  • project = PROJ AND created >= -7d - Ticket creati negli ultimi 7 giorni
  • assignee = currentUser() - Ticket assegnati a te
